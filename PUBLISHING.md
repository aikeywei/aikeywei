# GitHub Profile 发布说明

## 生效条件

GitHub 个人主页 README 需要放在一个特殊仓库里：

- 仓库名必须和用户名完全一致：`aikeywei`
- 仓库必须是 Public
- 仓库根目录需要有 `README.md`

也就是说，最终结构应该是：

```text
aikeywei/
  README.md
  assets/
    profile-banner.svg
```

## 推荐发布步骤

1. 打开 <https://github.com/new>。
2. Repository name 填 `aikeywei`。
3. Visibility 选择 Public。
4. 勾选 Add a README file，然后创建仓库。
5. 将本目录里的 `README.md` 和 `assets/profile-banner.svg` 放入这个仓库并提交。
6. 回到 <https://github.com/aikeywei>，刷新后就能看到新的主页。

## 可选微调

- 如果你希望更商业化：把 About 部分改成更偏“业务分析 / 系统建设 / 经营报表”的表达。
- 如果你希望更技术化：增加项目架构图、技术栈和最近正在构建的工具。
- 如果你希望更个人化：补充职业经历、博客、邮箱、公众号或个人网站入口。
